# price-tracker-bot

Self-hosted Telegram bot for multi-site price tracking with auto-quarantine, structured observability, and a plugin architecture for adding sites.

> **Status:** v0.1.0-dev — pre-public. Documentation will be expanded before first release.

## Quick start (placeholder — full docs in v0.1.0)

```bash
cp .env.example .env
# fill TELEGRAM_BOT_TOKEN and ALLOWED_USERS in .env
docker compose up -d
```
